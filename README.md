## 3️⃣ 💻 Smart Memory Manager & OS Simulator

### 🔍 Description

A **low-level memory simulator** that mimics OS memory allocation.

### ⚙️ Features

* First Fit, Best Fit, Worst Fit algorithms
* Process-based memory allocation
* Fragmentation detection
* Memory visualization
* Garbage collection simulation

---

### ▶️ How to Run

```bash
cd MemorySimulator
g++ *.cpp -o memory
.\memory.exe
```

---

### 🧠 Concepts Used

✔ Dynamic Memory (new/delete)
✔ Polymorphism (allocation strategies)
✔ Encapsulation (memory blocks)
✔ System-level logic

# ⚠️ Common Errors & Fixes

### ❌ Error: multiple definition of main

👉 Delete:

```bash
tempCodeRunnerFile.cpp
```

---

### ❌ Error: undefined reference

👉 Fix:

```bash
g++ *.cpp -o app
```

---

### ❌ Error: g++ not recognized

👉 Install MinGW / MSYS2 and add to PATH
